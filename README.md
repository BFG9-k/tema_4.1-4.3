удельной стоимости (value/weight).

- Считать предметы как объекты с весом и стоимостью

Входные данные:
Предметы: {(вес=2, стоимость=10), (вес=3, стоимость=20),
 (вес=4, стоимость=15), (вес=5, стоимость=25)}
