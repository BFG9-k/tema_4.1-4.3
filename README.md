Практическое задание (вариант 3). Дробный рюкзак с жадным алгоритмом.

Задача: реализовать жадный алгоритм для дробного рюкзака. Сортировка предметов по
удельной стоимости (value/weight).

Требования:
- Считать предметы как объекты с весом и стоимостью
- Вместимость рюкзака задается как параметр
- Вывести максимальную стоимость и набор предметов

Входные данные:
Предметы: {(вес=2, стоимость=10), (вес=3, стоимость=20),
 (вес=4, стоимость=15), (вес=5, стоимость=25)}
Вместимость: 8

Объяснение работы алгоритма:
1.
Вычисляем удельную стоимость каждого предмета путем деления стоимости на вес (value / weight):
a.value / a.weight > b.value / b.weight;

2. 
Сортируем предметы по убыванию удельной стоимости. Чем больше удельная стоимость, тем выгоднее включать этот предмет первым.
sort(items.begin(), items.end(), compare);

4.
Последовательно добавляем предметы в рюкзак согласно сортировке, пока позволяет оставшаяся вместимость рюкзака. Если какой-либо предмет целиком добавить нельзя, берем его долю, соответствующую остаточной емкости рюкзака.

totalValue += item.value;
capacity -= item.weight;
totalValue += item.value * ((double)capacity / item.weight);

Контрольный вопрос 3.

PTAS и FPTAS

Объясните различия между PTAS (Polynomial-Time Approximation Scheme) и FPTAS (Fully
Polynomial-Time Approximation Scheme).

Ответ:
PTAS — полиномиальная схема аппроксимации, время работы полиномиально от размера входа, но экспоненциально зависит от требуемой точности ε

FPTAS — полностью полиномиальная схема аппроксимации, время работы полиномиально как от размера входа, так и от обратного значения точности 1/ε
